# HR-Analytics-Dashboard

**Project Overview**

This project showcases an interactive HR analytics dashboard built using Tableau.
It analyzes employee data including hiring trends, attrition, salary distribution, and workforce demographics.
The dataset is synthetically generated using Python’s Faker library to simulate real-world HR scenarios.

**Problem Statement**

Organizations struggle to understand workforce trends due to scattered data.
This dashboard centralizes HR metrics to provide clear, data-driven insights for decision-making.

**Dataset Details**
	•	Generated using Faker (Python library)
	•	Contains 8000+ employee records
	•	No real data used (fully synthetic)

Key Fields:
	•	Age
	•	Gender
	•	Department
	•	Salary
	•	Education
	•	Employment Status

**Tech Stack**
	•	Tableau (Visualization)
	•	Python (Data Generation)
	•	Faker Library
	•	CSV

  **Dashboard Highlights**
  
🔹 Workforce Overview
	•	Active Employees KPI
	•	Hired vs Terminated Comparison
	•	Department Distribution

🔹 Demographics Analysis
	•	Gender Distribution
	•	Age Group Insights
	•	Education Breakdown

🔹 Salary Insights
	•	Salary vs Age Trend
	•	Salary by Education
	•	Gender Pay Comparison

🔹 Detailed Records
	•	Employee-level table
	•	Department, role, salary, and status

  **Key Insights**
	•	Sales and IT departments have the highest workforce concentration
	•	Higher education levels correlate with higher salary brackets
	•	Attrition is uneven across departments
	•	Salary trends increase with age and experience
